Fun With Numbers

This little game will take a number and will change one or more
of the integers between zero and that number as follows:

  • If the number is evenly divisible by three, it will return a "sorry" message.
  • If the number contains a '1', it will return a fun message.
  • If the number contains a '0', it will return a different fun message.
  • Otherwise, it will return the number.
You can specify in which direction you want the integers to be processed, too, and the results will be displayed in that direction.

Note that decimal numbers will be rounded and negative numbers will be made positive.
